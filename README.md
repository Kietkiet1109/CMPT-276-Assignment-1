# CMPT 276 Assignment 1 - Weather Finder App 

A simple, static web application hosted on Spring Boot that allows users to check current weather conditions for major cities. This project was built to demonstrate Git workflows, branching strategies, and parallel development conflict resolution.

**Deployed URL:** https://cmpt-276-assignment-1-jjfh.onrender.com/

---

## 1. Features

1.  **City Search:** Users can input any major city name (e.g., "Vancouver") to fetch real-time weather data.
2.  **Top 10 Cities:** A pre-defined list of famous global cities allows for one-click weather checks.
3.  **Real-Time Data:** Fetches Temperature, Wind Speed, and Humidity using the [Open-Meteo API](https://open-meteo.com/).
4.  **Responsive UI:** Clean, responsive design implemented with raw CSS (no frameworks).

---

## 2. Technologies Used

* **Backend:** Java Spring Boot (Static Resource Serving only)
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **API:** Open-Meteo (Geocoding & Forecast)
* **Deployment:** Docker, Render
* **Tools:** Git, GitHub
